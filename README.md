# Cryptocurrency API Data Analysis

## Overview

This project uses the CoinMarketCap API to collect real-time cryptocurrency market data and perform data analysis using Python.

The workflow includes:

- API integration with CoinMarketCap
- Automated data collection
- Data transformation and cleaning using Pandas
- CSV data storage
- Trend analysis of cryptocurrency percentage changes
- Data visualization using Seaborn and Matplotlib

## Technologies Used

- Python
- Pandas
- NumPy
- CoinMarketCap API
- Seaborn
- Matplotlib
- Jupyter Notebook

## Project Workflow

1. Connected to CoinMarketCap API.
2. Retrieved cryptocurrency market data.
3. Converted JSON responses into DataFrames.
4. Stored data in CSV format.
5. Cleaned and transformed the data.
6. Analyzed percentage changes over multiple time periods.
7. Created visualizations for trend comparison.

## Key Analysis

- 1 Hour Percentage Change
- 24 Hour Percentage Change
- 7 Day Percentage Change
- 30 Day Percentage Change
- 60 Day Percentage Change
- 90 Day Percentage Change

## Results

The project highlights cryptocurrency performance trends and enables comparison of short-term and long-term market movements.

## Note

API keys have been removed for security purposes. Users can generate their own CoinMarketCap API key and replace the placeholder value before running the notebook.
